# personal website
[https://kosuke-ukita.github.io/](https://kosuke-ukita.github.io/)

Built with Nuxt 3 + Tailwind CSS. Supports English/Japanese switching and dark mode.

## Pages

- **Home** — Short bio, recent news, selected publications, grants, awards, and research interests. Each section links to its dedicated page.
- **Biography** — Two versions of a biographical narrative: a short summary and an extended version covering educational background, research history, and current interests.
- **CV** — Education, work experience, publications, grants, awards, and skills. Includes a PDF download link and a line-wrap toggle for wide content.
- **Publications** — Full publication list grouped by year. Each entry shows venue, presentation type (Spotlight / Oral / Poster), refereed status, and links (paper, code, etc.). BibTeX can be copied from a modal.
- **Grants** — List of research grants with organization and amount.
- **Awards** — List of academic awards with organization and year.
- **News** — Chronological list of updates and announcements.
- **Contact** — Email addresses and office location.
