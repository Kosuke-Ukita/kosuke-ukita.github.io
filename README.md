# personal website
[https://kosuke-ukita.github.io/](https://kosuke-ukita.github.io/)

- [x] Built with Nuxt 3
- [x] Tailwind CSS
- [x] TypeScript
- [x] English/Japanese switching
- [x] light/dark mode switching

## Pages

**Home**
- short bio
- recent news
- selected publications
- selected grants
- selected awards
- research interests

Each section links to its dedicated page.

---

**Biography**

Two versions of a biographical narrative:
- short summary
- extended version covering educational background, research history, and current interests

---

**CV**

- education
- work experience
- publications
- grants
- awards
- skills

Includes a PDF download link and a line-wrap toggle for wide content.

---

**Publications**

Full publication list grouped by year.
Each entry shows:
- venue
- presentation type (Spotlight / Oral / Poster)
- refereed status
- links (paper, code, etc.)
- BibTeX can be copied from a modal

---

**Grants**

List of research grants with organization and amount.

---

**Awards**

List of academic awards with organization and year.

---

**News**

Chronological list of updates and announcements.

---

**Contact**

Email addresses and office location.
